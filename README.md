# Curated Notes

A clean, personal notes website for publishing organized, readable, and visually polished notes.

## Overview

Curated Notes is a custom notes platform built for storing and presenting written content in a simple, searchable, and professional format. The project is designed around clarity, fast navigation, and a polished reading experience.

## Features

- Organized note content
- MDX-based writing structure
- Clean reading interface
- Custom notes branding
- Responsive design
- Reusable UI components

## Tech Stack

- Next.js
- TypeScript
- MDX
- Tailwind CSS
- CSS Modules / Global Styles
